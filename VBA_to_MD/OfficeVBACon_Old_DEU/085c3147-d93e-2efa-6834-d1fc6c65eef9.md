
# Font2.NameAscii-Eigenschaft (Office)

Ruft die Schriftart ab oder legt sie fest, die für westlichen Text (Zeichen mit den Zeichencodes von 0 (null) bis 127) verwendet wird. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **NameAscii**

 _Ausdruck_ Ein Ausdruck, der ein **Font2** -Objekt zurückgibt.


### Rückgabewert

String
