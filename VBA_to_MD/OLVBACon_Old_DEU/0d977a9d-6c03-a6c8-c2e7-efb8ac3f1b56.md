
# TaskRequestUpdateItem.Body Property (Outlook)

Zurückgeben oder Festlegen einer  **Zeichenfolge** zurück, den Clear-Textkörper des Outlook-Elements darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Body**

 _Ausdruck_ Eine Variable, die ein **TaskRequestUpdateItem** -Objekt darstellt.
