
# WeekDay.Shift5 Property (Project)

Ruft ein  **[Shift](bf224646-d1c6-bc4a-1cce-a08b2f4e417d.md)** -Objekt zurück, die fünfte Schicht in einen Wochentag darstellt. Read-only **UMSCHALT**.


## Syntax

 _Ausdruck_. **Shift5**

 _Ausdruck_ Eine Variable, die ein **WeekDay** -Objekt darstellt.


## Beispiel

Im folgenden Beispiel wird für Freitage ein halber Arbeitstag festgelegt, indem eine Schicht von 08:00 Uhr bis 12:00 Uhr erstellt wird.


```
Sub HalfDayFridays() 
 
 With ActiveProject.Calendar.WeekDays(pjFriday) 
 .Shift1.Start = #8:00:00 AM# 
 .Shift1.Finish = #12:00:00 PM# 
 .Shift2.Clear 
 .Shift3.Clear 
 End With 
 
End Sub
```
