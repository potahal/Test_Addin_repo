
# ContactItem.SelectedMailingAddress Property (Outlook)

Gibt eine  **[OlMailingAddress](1a0822be-3c4c-f162-8ab7-12edb20a0949.md)** -Konstante zurück (oder legt diese fest), die den Typ der Postanschrift für den Kontakt angibt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **SelectedMailingAddress**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt.
