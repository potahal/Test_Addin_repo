
# OlkComboBox.RemoveItem Method (Outlook)

Entfernt das angegebene Element aus der Liste.


## Syntax

 _Ausdruck_. **RemoveItem**( ** _Index_** )

 _Ausdruck_ Eine Variable, die ein **OlkComboBox** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Long**|Ein nullbasierter Wert, der im Array mit Elementen in der Liste einen Index erstellt.|

## Hinweise

Wenn sich  _Index_ außerhalb des Bereichs der zulässigen Werte befindet (zwischen Null und **[ListCount](04fd21e1-e822-cf5f-a6ea-7c318778a163.md)** -1), wird eine entsprechende Fehlermeldung zurückgegeben.
