
# DocumentProperty.Application-Eigenschaft (Office)

Ruft ein  **Application** -Objekt, das der containeranwendung für das **DocumentProperty** -Objekt darstellt (Sie können diese Eigenschaft mit **einem Automatisierungsobjekt** verwenden containeranwendung für dieses Objekt zurückgeben). Schreibgeschützt.


## Syntax

 _Ausdruck_. **Application**( ** _ppidisp_** )

 _Ausdruck_ Eine Variable, die ein **DocumentProperty** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _ppidisp_|Erforderlich|**Objekt**|Stellt den Bezeichner für eine Dokumenteigenschaft dar.|

### Rückgabewert

[HRESULT]
