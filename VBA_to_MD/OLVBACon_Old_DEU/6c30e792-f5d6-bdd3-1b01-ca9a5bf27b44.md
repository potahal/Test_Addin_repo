
# ContactItem.BusinessTelephoneNumber Property (Outlook)

Zurückgeben oder Festlegen einer  **Zeichenfolge**, die die erste geschäftliche Telefonnummer für den Kontakt darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **BusinessTelephoneNumber**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt.
