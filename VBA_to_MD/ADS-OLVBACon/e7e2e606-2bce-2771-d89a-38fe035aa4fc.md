# ViewCtl.AdvancedFind メソッド (Outlook ビュー コントロール)




Microsoft Outlook の [**高度な検索**] ダイアログ ボックスを表示します。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**AdvancedFind**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ViewCtl** オブジェクトを表す変数を指定します。
