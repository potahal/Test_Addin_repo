
# ShapeRange.Connector Property (Project)
Ruft einen Wert, der angibt, ob der Formenbereich um eine Verbindung handelt. Read-only  **[MsoTriState](http://msdn.microsoft.com/en-us/library/office/ff860737%28v=office.15%29)**

## Syntax

 _Ausdruck_. **Connector**

 _Ausdruck_ Eine Variable, die ein ShapeRange **ShapeRange**-Objekt darstellt.


## Eigenschaftswert

 **MSOTRISTATE**
