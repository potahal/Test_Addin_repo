
# Application.WebSetStartPage Method (Project)

Gibt ein Dokument als Startseite in Microsoft Internet Explorer an.


## Syntax

 _Ausdruck_. **WebSetStartPage**( ** _Address_** )

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Address_|Optional|**String**|Die Adresse, die als Startseite verwenden. Wenn  **nicht angegeben wird,** wird das aktuelle Dokument als Startseite festgelegt.|
