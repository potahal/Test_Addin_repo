
# ContactItem.ForwardAsVcard Method (Outlook)

Erstellt ein  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** und fügt die Kontaktinformationen in vCard-Format an.


## Syntax

 _Ausdruck_. **ForwardAsVcard**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt.


### Rückgabewert

Ein  **MailItem** -Objekt, das das neue e-Mail-Element darstellt, das die Kontaktinformationen angefügt werden.


## Bemerkungen

vCard ist der Internetstandard zum Erstellen und Freigeben von virtuellen Visitenkarten.
