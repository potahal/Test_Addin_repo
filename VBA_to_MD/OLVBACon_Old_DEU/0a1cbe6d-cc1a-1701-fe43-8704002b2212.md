
# BusinessCardView.CardSize Property (Outlook)

Gibt zurück oder legt einen  **Long** -Wert, der die Größe als Prozentsatz, der einer elektronischen Visitenkarte (EBC) in der Ansicht darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **CardSize**

 _Ausdruck_ Ein Ausdruck, der ein **BusinessCardView** -Objekt zurückgibt.


## Hinweise

Diese Eigenschaft kann auf einen Wert zwischen 20 und 100 festgelegt werden. Wenn Sie für diese Eigenschaft einen niedrigeren Wert als 20 angeben, wird die Eigenschaft auf  **20** festgelegt. Wenn Sie einen höheren Wert als 100 angeben, wird sie auf **100** festgelegt.
