
# Project.Calculate イベント (Project)

プロジェクトのスケジュールが再計算されるときに発生します。


## 構文

 _式_. **Calculate**( ** _pj_**, )

 _式_ **Project** オブジェクトを返すオブジェクト式を指定します。


### パラメーター



|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _pj_|必須|**Project**|再スケジュールされるプロジェクトを表します。|

### 戻り値

なし


## 注釈

Project のイベントは、プロジェクトが他のドキュメントまたはアプリケーションに埋め込まれている場合は発生しません。
