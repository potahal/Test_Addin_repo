# PropertyPages.Item メソッド (Outlook)(機械翻訳)




コレクションから **PropertyPage** オブジェクトを返します。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Item**(**Index**)




            UNRESOLVED_TOKEN_VAL(offexpression)
            **PropertyPages** オブジェクトを表す変数。

### パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|---|---|---|---|
|*Index*|
                        UNRESOLVED_TOKEN_VAL(offrequired)
                      |**バリアント型 (Variant)**|オブジェクトのインデックス番号、またはコレクションのオブジェクトの既定のプロパティに適合する値を指定します。|



### 戻り値
指定したオブジェクトを表す文字列型 ( **Object** ) の値を指定します。
