# CustomXMLPart.XML プロパティ (Office)




現在の**CustomXMLPart**オブジェクトの XML 表現を取得します。読み取り専用です。

>[!NOTE]
>
              UNRESOLVED_TOKEN_VAL(osoffdevnodtdsincustomxml)
            





## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**XML**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **CustomXMLPart** オブジェクトを表すオブジェクト式を指定します。

### 戻り値
文字列型 (String)
