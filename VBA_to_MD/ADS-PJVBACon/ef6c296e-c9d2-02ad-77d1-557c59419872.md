# Application.EnterpriseProjectDelete メソッド (Project)




Project Server データベースからプロジェクトを削除します。Project Professional でのみ使用可能です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**EnterpriseProjectDelete**(**Name**)




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Application** オブジェクトを表す変数です。

### パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|---|---|---|---|
|*Name*|
                        UNRESOLVED_TOKEN_VAL(offoptional)
                      |**String**|Project Server データベースから削除されるプロジェクトの名前です。|



### 戻り値
**ブール型 (Boolean)**
