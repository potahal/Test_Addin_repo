
# CustomXMLPrefixMapping.Prefix-Eigenschaft (Office)

Ruft das Präfix für ein  **CustomXMLPrefixMapping** -Objekt ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Prefix**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLPrefixMapping** -Objekt zurückgibt.


### Rückgabewert

String
