# Project.BeforePrint イベント (Project)




プロジェクトが印刷される前に発生します。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**BeforePrint**(**pj**, )




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Project** オブジェクトを返すオブジェクト式を指定します。

### パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|---|---|---|---|
|*pj*|
                        UNRESOLVED_TOKEN_VAL(offrequired)
                      |**Project**|印刷されるプロジェクトを表します。|



### 戻り値
なし





## 注釈
Project のイベントは、プロジェクトが他のドキュメントまたはアプリケーションに埋め込まれている場合は発生しません。
