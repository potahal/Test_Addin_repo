
# SendRuleAction.ActionType Property (Outlook)

Gibt eine Konstante,  **OlRuleActionRedirect** aus der **[OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** -Aufzählung, die den Typ der Aktion angibt, die von der Regel zurück. Schreibgeschützt.


## Syntax

 _Ausdruck_. **ActionType**

 _Ausdruck_ Eine Variable, die ein **SendRuleAction** -Objekt darstellt.


## Hinweise

Die Regelaktion zum Senden wird nur von Regeln für das Empfangen von Nachrichten unterstützt. Weitere Informationen finden Sie unter [Angeben von Regelaktionen](c5f83c81-0e01-38aa-5ec7-3932b4443e43.md).
