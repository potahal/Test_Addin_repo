
# Explorer.AccountSelector Property (Outlook)

Gibt ein  **[AccountSelector](846f176e-5680-a214-7624-75f3a524c989.md)** -Objekt zurück, das die Microsoft Office Backstage-Ansicht für das **[Explorer](026591e5-049f-503a-4166-34e6dbc225fb.md)** -Objekt darstellt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **AccountSelector**

 _Ausdruck_ Eine Variable, die ein **Explorer** -Objekt darstellt.
