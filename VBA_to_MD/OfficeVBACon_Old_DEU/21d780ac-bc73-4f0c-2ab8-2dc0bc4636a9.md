
# ThemeEffectScheme.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **ThemeEffectScheme** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **ThemeEffectScheme** -Objekt zurückgibt.


### Rückgabewert

Object
