
# SearchScopes.Count-Eigenschaft (Office)

Ruft ein  **Long** zurück, der die Anzahl der Elemente in der **SearchScopes** -Objekt angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein **SearchScopes** -Objekt darstellt.


### Rückgabewert

Long
