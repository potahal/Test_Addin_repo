# ContactItem.BusinessHomePage プロパティ (Outlook)(機械翻訳)




連絡先の勤務先の Web ページの URL を表す**文字列**を設定または返します。読み取り/書き込み。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**BusinessHomePage**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ContactItem** オブジェクトを表す変数を指定します。
