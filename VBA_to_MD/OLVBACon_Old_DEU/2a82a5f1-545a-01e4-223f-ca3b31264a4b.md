
# TaskRequestAcceptItem.Read Event (Outlook)

Tritt auf, wenn eine Instanz des übergeordneten Objekts vom Benutzer zum Bearbeiten geöffnet wird.


## Syntax

 _Ausdruck_. **Read**

 _Ausdruck_ Eine Variable, die ein **TaskRequestAcceptItem** -Objekt darstellt.


## Bemerkungen

Das  **Read** -Ereignis unterscheidet sich von der **[Open](4a8d97b6-5502-d362-388b-de70174a0816.md)** -Ereignis, dass **eintritt, wenn der Benutzer das Element in einer Ansicht auswählt, dass in der Zelle bearbeiten ermöglicht, und wenn das Element in einem  **[Inspektor](d7384756-669c-0549-1032-c3b864187994.md)** geöffnet wird**.
