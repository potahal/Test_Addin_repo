
# CommandBarComboBox.Priority-Eigenschaft (Office)

Dient zum Abrufen oder Festlegen der Priorität von einem  **CommandBarComboBox** -Steuerelement. Priorität eines Steuerelements bestimmt, ob das Steuerelement aus einer angedockten Befehlsleiste entfernt werden kann, wenn die Befehlsleisten-Steuerelemente nicht in einer einzelnen Zeile passen. Lese-/Schreibzugriff.


## 


 **Hinweis**  "CommandBar"-Objekte wurden in einigen Microsoft Office-Anwendungen durch die neue Menüband-Komponente der Microsoft Office Fluent-Benutzeroberfläche abgelöst. Suchen Sie in der Hilfe nach dem Stichwort "Menüband", um weitere Informationen zu erhalten.


## Syntax

 _Ausdruck_. **Priority**

 _Ausdruck_ Eine Variable, die ein **[CommandBarComboBox](fcfe6bde-dea0-f1f1-ad30-d0e28f97dd07.md)** -Objekt darstellt.


## Bemerkungen

Gültige Werte für die Priorität sind 0 (null) bis 7, wobei der Standardwert  **3** lautet. Die Priorität 1 gibt an, dass das Steuerelement nicht aus einer Symbolleiste entfernt werden kann. Andere Prioritätswerte werden ignoriert.

Die  **Priority** -Eigenschaft wird nicht von Befehlsleisten-Steuerelemente verwendet, die Menübefehle sind.
