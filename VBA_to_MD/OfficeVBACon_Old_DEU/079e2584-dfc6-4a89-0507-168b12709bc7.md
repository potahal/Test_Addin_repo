
# SharedWorkspaceTask.Application-Eigenschaft (Office)

Ruft ein  **Application** -Objekt, das der containeranwendung für das **SharedWorkspaceTask** -Objekt darstellt (Sie können diese Eigenschaft mit **einem Automatisierungsobjekt** verwenden containeranwendung für dieses Objekt zurückgeben). Schreibgeschützt.


 **Hinweis**  Ab Microsoft Office 2010 ist dieses Objekt oder Element veraltet und sollte nicht verwendet werden.


## Syntax

 _Ausdruck_. **Application**

 _Ausdruck_ Eine Variable, die ein **SharedWorkspaceTask** -Objekt darstellt.


### Rückgabewert

Object
