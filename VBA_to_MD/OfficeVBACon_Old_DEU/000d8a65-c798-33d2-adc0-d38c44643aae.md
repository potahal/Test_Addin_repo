
# RulerLevels2.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **RulerLevels2** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **RulerLevels2** -Objekt zurückgibt.


### Rückgabewert

Object
