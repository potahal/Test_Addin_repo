
# TextRange2.LtrRun-Methode (Office)

Gibt ein  **TextRange2** -Objekt, das die angegebene Untermenge von links-nach-rechts Textläufe darstellt. Einen Textlauf besteht aus einem Bereich von Zeichen, die dieselben Schriftattribute aufweisen.


## Syntax

 _Ausdruck_. **LtrRun**

 _Ausdruck_ Ein Ausdruck, der ein **TextRange2** -Objekt zurückgibt.
