
# DistListItem.PropertyAccessor Property (Outlook)

Es wird ein  **[PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** -Objekt zurückgegeben, das das Erstellen, Abrufen, Festlegen und Löschen von Eigenschaften des übergeordneten **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** -Objekts unterstützt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **PropertyAccessor**

 _Ausdruck_ Eine Variable, die ein **DistListItem** -Objekt darstellt.
