
# ShapeRange.Count Property (Project)
Ruft die Anzahl der Formen im Formbereich an. Read-only  **Long**.

## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein ShapeRange **ShapeRange**-Objekt darstellt.


## Eigenschaftswert

 **INT**
