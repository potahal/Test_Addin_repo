
# Resource.Baseline6BudgetCost Property (Project)

Ruft das geplante Kostenbudget6 einer  **Ressource** ab, oder legt dieses fest. Wert vom Typ **Variant** mit Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Baseline6BudgetCost**

 _Ausdruck_ Eine Variable, die ein **Resource** -Objekt darstellt.
