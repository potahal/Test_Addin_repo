
# OlkTimeZoneControl.AppointmentTimeField Property (Outlook)

Dient zum Zurückgeben oder Festlegen einer  **[OlAppointmentTimeField](630706ba-c98a-e6de-73ce-c7071e0d1db2.md)** -Konstanten, die das Uhrzeitfeld für den Termin angibt, an den das Steuerelement gebunden ist. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **AppointmentTimeField**

 _Ausdruck_ Eine Variable, die ein **OlkTimeZoneControl** -Objekt darstellt.


## Hinweise

Dieses Steuerelement unterstützt keine Standarddatenbindung.
