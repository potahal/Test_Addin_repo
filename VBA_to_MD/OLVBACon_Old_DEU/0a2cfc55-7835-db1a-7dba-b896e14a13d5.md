
# ContactItem.BusinessCardLayoutXml Property (Outlook)

Zurückgeben oder Festlegen einer  **Zeichenfolge**, die das XML-Markup für das Layout der elektronischen Business Karte Lese-/Schreibzugriff darstellt.


## Syntax

 _Ausdruck_. **BusinessCardLayoutXml**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt


## Hinweise

Weitere Informationen zum XML-Schema für elektronische Visitenkarten finden Sie in der Microsoft Outlook 2010 XML-Schemareferenz in der [MSDN Library](http://msdn.microsoft.com/library).
