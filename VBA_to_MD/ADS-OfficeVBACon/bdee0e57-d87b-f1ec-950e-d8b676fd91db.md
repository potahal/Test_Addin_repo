# IConverter.HrGetFormat メソッド (Office)




この機能は、Open XML 形式の SDK でのみ使用できます。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**HrGetFormat**(**bstrPath**, **pbstrClass**, **pcap**, **ppcp**, **pcuic**)




            UNRESOLVED_TOKEN_VAL(offexpression)
            **IConverter** オブジェクトを返すオブジェクト式を指定します。

|**名前**|**必須/オプション**|**データ型**|**説明**|

                        UNRESOLVED_TOKEN_VAL(offrequired)
                      |**文字列型 (String)**||
