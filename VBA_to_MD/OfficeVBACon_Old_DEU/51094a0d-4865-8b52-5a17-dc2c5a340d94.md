
# CustomXMLValidationError.Text-Eigenschaft (Office)

Ruft den Text in das  **CustomXMLValidationError** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Text**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLValidationError** -Objekt zurückgibt.


### Rückgabewert

String
