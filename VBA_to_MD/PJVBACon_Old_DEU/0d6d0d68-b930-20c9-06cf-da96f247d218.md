
# WorkWeeks.Item Property (Project)

Ruft ein einzelnes  **WorkWeek** -Objekt aus der **WorkWeeks** -Auflistung ab. Read-only **WorkWeek**.


## Syntax

 _Ausdruck_. **Item**( ** _Index_** )

 _Ausdruck_ Ein Ausdruck, der ein **WorkWeeks** -Objekt zurückgibt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Variant**|Der Name oder die Indexnummer des zurückgebenden Objekts.|
