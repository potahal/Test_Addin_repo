
# FormRegion.Language Property (Outlook)

Gibt eine  **lange**, der die LCID für die aktuelle Sprache angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Language**

 _Ausdruck_ Eine Variable, die ein **FormRegion** -Objekt darstellt.
