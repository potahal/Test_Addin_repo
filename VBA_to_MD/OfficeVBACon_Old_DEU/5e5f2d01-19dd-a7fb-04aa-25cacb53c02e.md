
# SharedWorkspaceLink.Save-Methode (Office)

Lädt programmgesteuert ausgeführte Änderungen auf einen freigegebenen Server hoch.


 **Hinweis**  Ab Microsoft Office 2010 ist dieses Objekt oder Element veraltet und sollte nicht verwendet werden.


## Syntax

 _Ausdruck_. **Save**

 _Ausdruck_ Eine Variable, die ein **SharedWorkspaceLink** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _bstrQueryName_|Erforderlich|**String**|Der Name der Abfrage, mit der die Verknüpfung des freigegebenen Arbeitsbereichs geändert wird.|

## Bemerkungen

Verwenden Sie die  **Save** -Methode, um Änderungen auf den Server hochzuladen, nachdem Eigenschaften einer Verknüpfung eines freigegebenen Arbeitsbereichs geändert wurden.
