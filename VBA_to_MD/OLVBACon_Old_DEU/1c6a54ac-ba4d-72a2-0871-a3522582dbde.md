
# Table.Parent Property (Outlook)

Gibt das übergeordnete  **Objekt** des **[Table](0affaafd-93fe-227a-acee-e09a86cadc20.md)** -Objekts zurück. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Eine Variable, die ein **Table** -Objekt darstellt.
