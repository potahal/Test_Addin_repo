
# TaskItem.TotalWork Property (Outlook)

Zurückgeben oder Festlegen einer  **Long**, der angibt, der gesamten Arbeit für den Vorgang. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **TotalWork**

 _Ausdruck_ Eine Variable, die ein **TaskItem** -Objekt darstellt.


## Bemerkungen

 **TotalWork** entspricht dem Feld **Gesamtaufwand** auf der Registerkarte **Details** eines Aufgabenelements. Es wird in Einheiten von Minuten gespeichert. Feld **Gesamtaufwand** auf dem standard Aufgabenformular wird an die **TotalWork (** Eigenschaft) gebunden. Standardmäßig wird das Feld ein 8-Stunden-Tag und 40 Stunden Woche vorausgesetzt.
