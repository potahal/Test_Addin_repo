
# RemoteItem.TransferTime Property (Outlook)

Gibt ein  **Long** zurück, der die Übertragungsdauer (in Sekunden) für das Remoteelement angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **TransferTime**

 _Ausdruck_ Eine Variable, die ein **RemoteItem** -Objekt darstellt.
