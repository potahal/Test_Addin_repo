# ThemeFontScheme オブジェクト (Office)




Microsoft Office のテーマのフォント設定を表します。

## 例
次の例では、Microsoft Office テーマのフォント設定を表すオブジェクトを作成します。

```vba
Dim tTheme As OfficeTheme 
Dim tfsThemeFontScheme As ThemeFontScheme 
Set tfsThemeFontScheme = tTheme.ThemeFontScheme 

```
