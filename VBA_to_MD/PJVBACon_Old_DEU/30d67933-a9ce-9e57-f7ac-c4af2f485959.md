
# Cell.CellColor Property (Project)

Dient zum Abrufen oder Festlegen der Hintergrundfarbe der Zelle. Lese-/Schreibzugriff  **PjColor**.


## Syntax

 _Ausdruck_. **CellColor**

 _Ausdruck_ Eine Variable, die ein **Cell** -Objekt darstellt.


## Hinweise

Die  **CellColor** -Eigenschaft kann eine der folgenden **[PjColor](46108cf5-1e35-9774-b424-6c84223d9aac.md)** -Konstanten sein:
