
# WorkflowTemplate.Name-Eigenschaft (Office)

Ruft den Namen des  **WorkflowTemplate** -Objekts ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Name**

 _Ausdruck_ Ein Ausdruck, der ein **WorkflowTemplate** -Objekt zurückgibt.


### Rückgabewert

String
