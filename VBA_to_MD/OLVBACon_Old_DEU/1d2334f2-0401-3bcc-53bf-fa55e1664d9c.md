
# ContactItem.HomeAddressCity Property (Outlook)

Zurückgeben oder Festlegen einer  **Zeichenfolge**, der die Stadt der Privatadresse für den Kontakt darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **HomeAddressCity**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt.


## Bemerkungen

Diese Eigenschaft wird durch Analysieren aus der  **[HomeAddress (](c7ba836b-4b55-cedb-35f6-e6540bdf2c58.md)** Eigenschaft), aber möglicherweise geändert oder unabhängig eingegeben werden, wenn Sie nicht richtig analysiert wurde. Beachten Sie, dass es sich bei einer solchen geändert oder Einträge für diese Eigenschaft von nachfolgende Änderung oder Einträge **HomeAddress** überschrieben werden.
