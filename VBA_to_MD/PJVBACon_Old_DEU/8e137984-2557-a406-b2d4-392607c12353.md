
# Group.Parent Property (Project)

Gibt das übergeordnete Objekt des  **Group** -Objekt zurück. Nur-Lese **Projekt**.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Eine Variable, die ein **Group** -Objekt darstellt.


## Bemerkungen

Verwenden Sie die  **Parent** -Eigenschaft, um die Eigenschaften oder Methoden des übergeordneten Elements eines Objekts zuzugreifen.
