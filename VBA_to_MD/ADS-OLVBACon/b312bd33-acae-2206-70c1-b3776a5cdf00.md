# ViewCtl.CustomizeView メソッド (Outlook ビュー コントロール)




Microsoft Outlook の [**ビュー設定の概要**] ダイアログ ボックスを表示します。



## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**CustomizeView**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ViewCtl** オブジェクトを表す変数を指定します。



## 解説
このメソッドによって、ユーザーはコントロールの現在のビューをカスタマイズできます。
