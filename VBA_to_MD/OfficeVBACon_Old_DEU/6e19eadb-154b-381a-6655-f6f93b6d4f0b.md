
# CustomXMLPart.Id-Eigenschaft (Office)

Ruft eine  **Zeichenfolge** mit der GUID, die auf das aktuelle **CustomXMLPart** -Objekt zugewiesen. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Id**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLPart** -Objekt zurückgibt.


### Rückgabewert

String
