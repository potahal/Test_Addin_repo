
# PostItem.PropertyAccessor Property (Outlook)

Es wird ein  **[PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** -Objekt zurückgegeben, das das Erstellen, Abrufen, Festlegen und Löschen von Eigenschaften des übergeordneten **[PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)** -Objekts unterstützt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **PropertyAccessor**

 _Ausdruck_ Eine Variable, die ein **PostItem** -Objekt darstellt.
