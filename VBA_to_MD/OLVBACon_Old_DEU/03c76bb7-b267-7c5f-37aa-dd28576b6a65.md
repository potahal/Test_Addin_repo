
# TaskRequestAcceptItem.BeforeAutoSave Event (Outlook)

Tritt auf, bevor das Element automatisch von Outlook gespeichert wird.


## Syntax

 _Ausdruck_. **BeforeAutoSave**( ** _Cancel_** )

 _Ausdruck_ Eine Variable, die ein **TaskRequestAcceptItem** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Cancel_|Erforderlich|**Boolean**|Festgelegt auf  **true fest,** um den Vorgang abzubrechen. anderenfalls auf **False** festgelegt, um die **[TaskRequestAcceptItem](a2905f72-0a67-b07d-7f85-84fe4de17c25.md)** zu speichernde zu erm√∂glichen.|
