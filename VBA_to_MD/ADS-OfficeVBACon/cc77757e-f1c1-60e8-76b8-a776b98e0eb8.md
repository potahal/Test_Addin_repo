# MetaProperty.Id プロパティ (Office)




**MetaProperty**オブジェクトの ID を取得します。読み取り専用です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Id**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **MetaProperty** オブジェクトを表すオブジェクト式を指定します。

### 戻り値
文字列型 (String)
