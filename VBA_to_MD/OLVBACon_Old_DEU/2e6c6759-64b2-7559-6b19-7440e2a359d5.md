
# PostItem.ExpiryTime Property (Outlook)

Zurück oder legt  **Datum**, der das Datum und die Uhrzeit, an dem das Element ungültig wird und gelöscht werden. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **ExpiryTime**

 _Ausdruck_ Eine Variable, die ein **PostItem** -Objekt darstellt.
