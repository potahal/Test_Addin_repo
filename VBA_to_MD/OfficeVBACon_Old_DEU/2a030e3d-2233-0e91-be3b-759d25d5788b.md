
# Permission.Item-Eigenschaft (Office)

Ruft ein  **UserPermission** -Objekt, das ein Mitglied der **Permission** -Auflistung ist. Das **UserPermission** -Objekt verbindet einen Satz von Berechtigungen für das aktive Dokument mit einem einzelnen Benutzer und einem optionalen Ablaufdatum. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Item**( ** _Index_** )

 _Ausdruck_ Erforderlich. Eine Variable, die ein **[Permission](4bdf7058-d4ba-0bd4-c5cd-141d67245ced.md)** -Objekt darstellt. Ein Element, das ein **Permission** -Objekt zurückgibt.
