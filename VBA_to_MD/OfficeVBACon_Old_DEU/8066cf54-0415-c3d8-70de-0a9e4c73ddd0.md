
# CustomXMLPrefixMappings.Creator-Eigenschaft (Office)

Ruft eine 32-Bit-Ganzzahl, die die Anwendung angibt, in der das  **CustomXMLPrefixMappings** -Objekt erstellt wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Creator**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLPrefixMappings** -Objekt zurückgibt.


### Rückgabewert

Long
