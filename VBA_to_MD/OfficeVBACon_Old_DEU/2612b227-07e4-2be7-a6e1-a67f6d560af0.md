
# ODSOFilters.Count-Eigenschaft (Office)

Ruft ein  **Long** zurück, der die Anzahl der Elemente in der **ODSOFilters** -Auflistung angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein **ODSOFilters** -Objekt darstellt.


### Rückgabewert

Long
