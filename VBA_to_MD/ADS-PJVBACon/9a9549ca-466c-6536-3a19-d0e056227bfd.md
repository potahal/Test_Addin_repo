# Selection.FieldIDList プロパティ (Project)




選択したフィールドのすべてのフィールド id 番号を表す**List**オブジェクトを取得します。読み取り専用**リスト**。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**FieldIDList**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Selection** オブジェクトを表す変数です。
