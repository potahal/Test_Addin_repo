
# ParagraphFormat2.Parent-Eigenschaft (Office)

Ruft ab, der übergeordnete Objekt für das  **ParagraphFormat2** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **ParagraphFormat2** -Objekt zurückgibt.


### Rückgabewert

Object
