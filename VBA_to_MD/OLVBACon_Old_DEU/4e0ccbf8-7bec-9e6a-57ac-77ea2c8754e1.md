
# NoteItem.Height Property (Outlook)

Zurück oder legt eine  **lange**, für die Höhe des Notizfensters (in Pixel). Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Height**

 _Ausdruck_ Eine Variable, die ein **NoteItem** -Objekt darstellt.
