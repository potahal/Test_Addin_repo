
# FileTypes.Item-Eigenschaft (Office)

Ruft einen Wert ab, der angibt, nach welchem Dateityp gesucht wird. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Item**( ** _Index_** )

 _Ausdruck_ Erforderlich. Eine Variable, die ein **[FileTypes](5e8b5240-5ebd-704d-72e6-1f4ad951dfdc.md)** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Long**|Die Indexzahl des zurückzugebenden Objekts.|
