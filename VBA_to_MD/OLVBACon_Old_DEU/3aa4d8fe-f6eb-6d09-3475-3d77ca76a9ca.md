
# MailItem.Parent Property (Outlook)

Gibt das übergeordnete  **Objekt** des angegebenen Objekts zurück. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Eine Variable, die ein **MailItem** -Objekt darstellt.
