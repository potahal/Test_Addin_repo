
# ODSOColumn.Value-Eigenschaft (Office)

Ruft den Wert eines Datenfelds in einer Seriendruck-Datenquelle ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Value**

 _Ausdruck_ Eine Variable, die ein **ODSOColumn** -Objekt darstellt.


### Rückgabewert

String
