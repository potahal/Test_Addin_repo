
# Resource.Baseline3BudgetWork Property (Project)

Ruft das geplante Arbeitsbudget3 einer  **Ressource** in Stunden ab, oder legt dieses fest. Wert vom Typ **Variant** mit Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Baseline3BudgetWork**

 _Ausdruck_ Eine Variable, die ein **Resource** -Objekt darstellt.
