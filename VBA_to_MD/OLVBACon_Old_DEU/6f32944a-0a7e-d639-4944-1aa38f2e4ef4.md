
# OlkComboBox.EnterFieldBehavior Property (Outlook)

Gibt eine  **[olEnterFieldBehavior](4f9271f9-32db-08c7-f452-12e9793d1f9b.md)** -Konstante zurück oder legt sie fest, mit der das Auswahlverhalten eines Steuerelements angegeben wird, wenn es den Fokus erhält. Lese-­/Schreibzugriff.


## Syntax

 _Ausdruck_. **EnterFieldBehavior**

 _Ausdruck_ Eine Variable, die ein **OlkComboBox** -Objekt darstellt.


## Hinweise

Der Standardwert ist  **OlEnterFieldBehaviorSelectAll**.
