
# OlkComboBox.ListIndex Property (Outlook)

Gibt, oder legt eine  **lange**, der den Speicherort des aktuell ausgewählten Elements in der Liste des Kombinationsfelds angibt steuern. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **ListIndex**

 _Ausdruck_ Eine Variable, die ein **OlkComboBox** -Objekt darstellt.


## Hinweise

 **ListIndex** ist ein nullbasierter Wert in das Array von Elementen in der Liste. Der Standardwert ist-1, was bedeutet, dass kein Element ausgewählt ist.
