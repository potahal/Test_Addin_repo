
# SharingItem.SenderEmailType Property (Outlook)

Gibt eine  **Zeichenfolge**, die den Typ des Eintrags für die e-Mail-Adresse für den Absender des **[SharingItem-Objekt](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**, z. B. "SMTP" für Internetwebsites darstellt Adresse, "EX" für eine Adresse für den Microsoft Exchange-Server, und so weiter. Schreibgeschützt.


## Syntax

 _Ausdruck_. **SenderEmailType**

 _Ausdruck_ Eine Variable, die ein **SharingItem** -Objekt darstellt.
