
# ODSOFilters.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt des **ODSOFilters** -Objekts. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Eine Variable, die ein **ODSOFilters** -Objekt darstellt.


### Rückgabewert

Object
