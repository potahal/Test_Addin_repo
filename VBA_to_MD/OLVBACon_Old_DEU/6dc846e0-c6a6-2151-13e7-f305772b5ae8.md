
# DistListItem.IsMarkedAsTask Property (Outlook)

Gibt einen  **booleschen** Wert, der angibt, ob das **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** als Aufgabe markiert ist. Schreibgeschützt.


## Syntax

 _Ausdruck_. **IsMarkedAsTask**

 _Ausdruck_ Ein Ausdruck, der ein **DistListItem** -Objekt zurückgibt.
