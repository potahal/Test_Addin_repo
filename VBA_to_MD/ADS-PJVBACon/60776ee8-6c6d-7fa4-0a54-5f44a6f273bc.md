# PjAccountType 列挙 (Project)




Project Server アカウントの種類を指定する定数を含みます。

|**名前**|**値**|**説明**|
|---|---|---|
|**pjManagerAccount**|1|管理者アカウント|
|**pjResourceAccount**|0|リソース アカウント|
