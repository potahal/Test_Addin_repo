
# Application.SaveCompletedToServer Event (Project)

Tritt auf, wenn Project Professional erfolgreich den Auftrag  **Projekt speichern** in der Project Server-Warteschlange platziert.


## Syntax

 _Ausdruck_. **SaveCompletedToServer**( ** _bstrName_**, ** _bstrprojGuid_** )

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _bstrName_|Erforderlich|**String**|Name des Projekts.|
| _bstrprojGuid_|Erforderlich|**String**|GUID des Projekts|
