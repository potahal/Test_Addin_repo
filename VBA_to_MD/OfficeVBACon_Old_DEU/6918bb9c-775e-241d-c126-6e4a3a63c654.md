
# SignatureSet.Count-Eigenschaft (Office)

Ruft ein  **Long** zurück, der die Anzahl der Elemente in der **SignatureSet** -Objekt angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein **SignatureSet** -Objekt darstellt.


### Rückgabewert

Long
