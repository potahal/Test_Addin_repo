# Account.CurrentUser プロパティ (Outlook)(機械翻訳)




アカウントの現在のユーザー ID を表す **Recipient** オブジェクトを返します。値の取得のみ可能です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**CurrentUser**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Account** オブジェクトを表す変数を指定します。
