
# CommandBars.GetScreentipMso-Methode (Office)

Gibt die QuickInfo des vom  **IdMso** -Parameter als Zeichenfolge zurück.


## 


 **Hinweis**  "CommandBar"-Objekte wurden in einigen Microsoft Office-Anwendungen durch die neue Menüband-Komponente der Microsoft Office Fluent-Benutzeroberfläche abgelöst. Suchen Sie in der Hilfe nach dem Stichwort "Menüband", um weitere Informationen zu erhalten.


## Syntax

 _Ausdruck_. **GetScreentipMso**( ** _idMso_** )

 _Ausdruck_ Ein Ausdruck, der ein **CommandBars** -Objekt zurückgibt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _idMso_|Erforderlich|**String**|ID für das Steuerelement.|

### Rückgabewert

String


## Beispiel

Im folgenden Beispiel wird der  **String** -Wert **Paste** zurückgegeben.


```
Application.CommandBars.GetScreentipMso("Paste")
```
