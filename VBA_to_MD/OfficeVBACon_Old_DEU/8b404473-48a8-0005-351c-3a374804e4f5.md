
# SmartArtColor.Creator-Eigenschaft (Office)

Gibt eine 32-Bit-Ganzzahl, die die Anwendung angibt, in der das Objekt erstellt wurde. Schreibgeschützt


## Syntax

 _Ausdruck_. **Creator**

 _Ausdruck_ Ein Ausdruck, der ein **SmartArtColor** -Objekt zurückgibt.
