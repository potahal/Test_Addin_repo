
# DocumentItem.BeforeAutoSave Event (Outlook)

Tritt auf, bevor das Element automatisch von Outlook gespeichert wird.


## Syntax

 _Ausdruck_. **BeforeAutoSave**( ** _Cancel_** )

 _Ausdruck_ Eine Variable, die ein **DocumentItem** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Cancel_|Erforderlich|**Boolean**|Festgelegt auf  **true fest,** um den Vorgang abzubrechen. anderenfalls auf **False** festgelegt, um die **[DocumentItem](7b0a6af0-6632-3ff6-841f-5b081d0d68d8.md)** zu speichernde zu erm√∂glichen.|
