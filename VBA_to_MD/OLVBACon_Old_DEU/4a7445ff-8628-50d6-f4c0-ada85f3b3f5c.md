
# Conflicts.Count Property (Outlook)

Gibt ein  **Long** zurück, der die Anzahl der Objekte in der angegebenen Auflistung angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein **Conflicts** -Objekt darstellt.
