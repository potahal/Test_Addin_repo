
# Application.EditRedo Method (Project)

Wiederholt das oberste Element in einem Redo-Stapel.


## Syntax

 _Ausdruck_. **EditRedo**( ** _fRedo_** )

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _fRedo_|Optional|**Integer**|Gibt die Anzahl von Elementen zu wiederholen. Wenn die Gesamtzahl der Elemente in der Redo-Stapel kleiner als fRedo ist, wiederholt  **EditRedo** alle Elemente.|
