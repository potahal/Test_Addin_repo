
# ThemeColor.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **ThemeColor** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **ThemeColor** -Objekt zurückgibt.


### Rückgabewert

Object
