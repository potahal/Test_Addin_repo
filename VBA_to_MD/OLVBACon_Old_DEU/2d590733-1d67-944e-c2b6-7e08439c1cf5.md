
# MailItem.ReplyRecipients Property (Outlook)

Gibt eine  **[Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** -Auflistung, die alle Antwortempfänger-Objekte für das Outlook-Element darstellt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **ReplyRecipients**

 _Ausdruck_ Eine Variable, die ein **MailItem** -Objekt darstellt.
