
# TimelineView.GroupByFields Property (Outlook)

Gibt ein  **[OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** -Objekt zurück, das den Satz von Feldern darstellt, nach denen die im **[TimelineView](fb14c1a1-f542-fa1e-f30f-c5ee3d2f0206.md)** -Objekt angezeigten Elemente gruppiert sind, bzw. legt dieses fest. Schreibgeschützt.


## Syntax

 _Ausdruck_. **GroupByFields**

 _Ausdruck_ Eine Variable, die ein **TimelineView** -Objekt darstellt.
