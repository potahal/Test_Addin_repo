
# SelectNamesDialog.Class Property (Outlook)

Es wird eine Konstante in der  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** -Aufzählung zurückgegeben, die die Klasse des **[SelectNamesDialog](1522736a-3cad-9f1c-4da9-b52a3a01731c.md)** -Objekts angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Class**

 _Ausdruck_ Eine Variable, die ein **SelectNamesDialog** -Objekt darstellt.
