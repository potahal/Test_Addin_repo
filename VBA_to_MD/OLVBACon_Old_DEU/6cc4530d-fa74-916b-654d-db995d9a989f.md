
# MailItem.IsMarkedAsTask Property (Outlook)

Gibt einen  **booleschen** Wert, der angibt, ob das **[MailItem-Objekt](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** als Aufgabe markiert ist. Schreibgeschützt.


## Syntax

 _Ausdruck_. **IsMarkedAsTask**

 _Ausdruck_ Ein Ausdruck, der ein **MailItem** -Objekt zurückgibt.
