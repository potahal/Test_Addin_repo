
# PeopleView.Standard Property (Outlook)
Gibt einen  **Boolean** -Wert, der angibt, ob das Objekt[PeopleView](7b569709-5da8-a950-a0fb-9d64b520a21b.md) eine integrierte Outlook-Ansicht ist. Schreibgeschützt.

## Versionsinformationen

Hinzugefügte Version: Outlook 2013


## Syntax

 _Ausdruck_. **Standard**

 _Ausdruck_ Eine Variable, die ein **PeopleView** -Objekt darstellt.


## Eigenschaftswert

 **BOOL**
