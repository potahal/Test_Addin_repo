
# DocumentLibraryVersion.Creator-Eigenschaft (Office)

Ruft eine 32-Bit-Ganzzahl, die die Anwendung angibt, in der das  **DocumentLibraryVersion** -Objekt erstellt wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Creator**

 _Ausdruck_ Eine Variable, die ein **DocumentLibraryVersion** -Objekt darstellt.


### Rückgabewert

Long
