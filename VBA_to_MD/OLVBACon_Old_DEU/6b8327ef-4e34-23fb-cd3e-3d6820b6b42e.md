
# SharingItem.RetentionPolicyName Property (Outlook)

Gibt eine  **Zeichenfolge**, die den Namen der Aufbewahrungsrichtlinie angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **RetentionPolicyName**

 _Ausdruck_ Eine Variable, die ein **SharingItem** -Objekt darstellt


## Hinweise

Aufbewahrung aktiviert und deaktiviert ist von einem Administrator für einen Exchange-Server auf einer Postfachebene. Das Feature ist verfügbar, nur auf einem Exchange-Postfach mit Messaging Records Management (MRM), Version 2.0 oder höher aktiviert. Ein Beispiel für einen Richtliniennamen Retention ist "Zeitintervall für Ablauf schnelle Suche definieren".
