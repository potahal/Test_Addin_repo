
# Resource.Enterprise Property (Project)

 **True,** Wenn die Ressource eine Enterprise-Ressource, **False,** Wenn eine lokale Ressource ist. Read-only **Boolean**.


## Syntax

 _Ausdruck_ Eine Variable, die ein **Resource** -Objekt darstellt.
