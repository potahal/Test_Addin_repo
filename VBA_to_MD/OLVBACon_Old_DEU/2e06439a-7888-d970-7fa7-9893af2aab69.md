
# OlkCheckBox.Enter Event (Outlook)

Tritt auf, wenn das Steuerelement unmittelbar nach der  **Exit** -Ereignis für das vorherige Steuerelement den Fokus erhält.


## Syntax

 _Ausdruck_. **Enter**

 _Ausdruck_ Eine Variable, die ein **OlkCheckBox** -Objekt darstellt.


## Hinweise

Dies ist ähnlich wie die Steuerelemente mit verwaltetem Code oder in Microsoft Visual Basic 6  **GotFocus** -Ereignis.
