
# FormRegion.Close Event (Outlook)

Stellt das Schließen des Rahmens für einen Formularbereich dar. Tritt direkt vor dem  **[Close](5a83b3d3-6096-9e37-88b1-00f97c0bf8bd.md)** -Ereignis des Inspektors auf, das dem Outlook-Element zugeordnet ist.


## Syntax

 _Ausdruck_. **Close**

 _Ausdruck_ Eine Variable, die ein **FormRegion** -Objekt darstellt.
