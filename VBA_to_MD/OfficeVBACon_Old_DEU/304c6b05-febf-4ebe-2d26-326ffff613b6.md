
# TextRange2.TrimText-Methode (Office)

Gibt ein **TextRange2** -Objekt, das den angegebenen Text darstellt, in das Leerzeichen entfernt wurde.


## Syntax

 _Ausdruck_. **TrimText**

 _Ausdruck_ Ein Ausdruck, der ein **TextRange2** -Objekt zurückgibt.


### Rückgabewert

TextRange2
