
# PjDayLabel Enumeration (Project)

Enthält Konstanten, mit denen das Anzeigeformat für Daten auf einer Zeitskala angegeben wird.



|**Name**|**Wert**|**Beschreibung**|
|:-----|:-----|:-----|
|**pjDayLabelDay_ddd**|19|Beispiele: Mon, Die|
|**pjDayLabelDay_dddd**|18|Beispiele: Montag, Dienstag|
|**pjDayLabelDay_ddi**|119|Beispiele: Mo, Di|
|**pjDayLabelDay_di**|20|Beispiele: M, D|
|**pjDayLabelNoDateFormat**|35|Es wird kein Datum angezeigt.|
