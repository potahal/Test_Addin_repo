
# TextRange2.Creator-Eigenschaft (Office)

Ruft eine 32-Bit-Ganzzahl, die die Anwendung angibt, in der das  **TextRange2** -Objekt erstellt wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Creator**

 _Ausdruck_ Ein Ausdruck, der ein **TextRange2** -Objekt zurückgibt.


### Rückgabewert

Long
