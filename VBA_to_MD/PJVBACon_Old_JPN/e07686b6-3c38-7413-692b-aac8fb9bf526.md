
# Application.Macro メソッド (Project)

マクロを実行します。


## 構文

 _式_. **Macro**( ** _Name_** )

 _式_ **Application** オブジェクトを表す変数です。


### パラメーター



|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _Name_|省略可能|**文字列型 (String)**|実行するマクロの名前です。 **名** を省略した場合、[ **マクロ** ] ダイアログ ボックスが表示されます。|

### 戻り値

 **ブール型 (Boolean)**


## 例

次の使用例は、「シフトの確認」というマクロを実行します。


```
Sub RunMacro() 
 Macro "CheckShifts" 
End Sub
```
