
# Filters Object (Project)

Enthält eine Auflistung von  **[Filter](abcd72a7-b86b-783e-16e0-f50a48b1fed2.md)** -Objekten.


## Beispiel

 **Verwenden der Filters-Auflistung**

Im folgenden Beispiel wird ein Filter für kritische Vorgänge auf das aktive Projekt angewendet.




```
ActiveProject.TaskFilters("Critical").Apply
```
