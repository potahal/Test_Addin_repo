# Resource.DefaultAssignmentOwner プロパティ (Project)




設定またはリソースに行った割り当ての進捗状況の更新を提供することを担当するユーザー名を取得します。読み取りまたは書き込み**文字列**です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**DefaultAssignmentOwner**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Resource** オブジェクトを表す変数です。



## 注釈
**DefaultAssignmentOwner**プロパティは、リソースのすべての割り当てに影響します。プロパティは、有効な Project Server ユーザーまたは**null**に設定する必要があります。



**DefaultAssignmentOwner**プロパティは、Project Professional でのみ使用可能です。
