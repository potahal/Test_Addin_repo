
# TaskItem.Ownership Property (Outlook)

Gibt eine  **[OlTaskOwnership](1b3f1b21-e928-4cc2-e6ae-81cf9db688ad.md)** zurück, die den Besitzstatus der Aufgabe angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Ownership**

 _Ausdruck_ Eine Variable, die ein **TaskItem** -Objekt darstellt.
