
# OlkTimeZoneControl.SelectedTimeZoneIndex Property (Outlook)

Dient zum Zurückgeben oder Festlegen eines Indexwerts in der  **[Application.TimeZones](920e55d1-9914-fa74-101a-921083328d23.md)** -Auflistung, der die ausgewählte Zeitzone bestimmt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **SelectedTimeZoneIndex**

 _Ausdruck_ Eine Variable, die ein **OlkTimeZoneControl** -Objekt darstellt.


## Hinweise

 **SelectedTimeZoneIndex** ist 1-basierte und ein Wert von null (0) bedeutet, dass keine Zeitzone ausgewählt wurde.
