
# CustomTaskPane.Application-Eigenschaft (Office)

Ruft das  **Application** -Objekt der Host-Anwendung ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Application**

 _Ausdruck_ Ein Ausdruck, der ein **CustomTaskPane** -Objekt zurückgibt.


### Rückgabewert

Object
