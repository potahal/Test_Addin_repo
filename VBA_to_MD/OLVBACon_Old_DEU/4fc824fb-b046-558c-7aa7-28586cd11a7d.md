
# AppointmentItem.Recipients Property (Outlook)

Gibt eine  **[Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** -Auflistung, die alle Empfänger für das Outlook-Element darstellt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Recipients**

 _Ausdruck_ Eine Variable, die ein **AppointmentItem** -Objekt darstellt.


## Bemerkungen

Ein Empfänger kann von einer Zeichenfolge festgelegt werden, die den Anzeigenamen des Empfängers, den Alias oder die vollständige SMTP-E-Mail-Adresse darstellt.
