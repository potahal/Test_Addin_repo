
# Font2.Creator-Eigenschaft (Office)

Ruft einen Wert ab, der die Anwendung angibt, in der das Objekt erstellt wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Creator**

 _Ausdruck_ Ein Ausdruck, der ein **Font2** -Objekt zurückgibt.


### Rückgabewert

Long
