
# Application.Version Property (Outlook)

Zurückgeben oder Festlegen einer  **Zeichenfolge**, die die Anzahl der Version. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Version**

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.
