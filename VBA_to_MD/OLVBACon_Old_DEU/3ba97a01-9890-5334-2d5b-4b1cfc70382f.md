
# SharingItem.Submitted Property (Outlook)

Gibt einen  **booleschen** Wert, der **True** ist, wenn das **[SharingItem-Objekt](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** gesendet wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Submitted**

 _Ausdruck_ Eine Variable, die ein **SharingItem** -Objekt darstellt


## Hinweise

Eine Nachricht wird immer in einem Ordner, gewöhnlich im Ordner  **Postausgang**, erstellt und gesendet.
