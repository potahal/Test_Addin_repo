
# Resource.Baseline10BudgetWork Property (Project)

Ruft das geplante Arbeitsbudget10 einer  **Ressource** in Stunden ab, oder legt dieses fest. Wert vom Typ **Variant** mit Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Baseline10BudgetWork**

 _Ausdruck_ Eine Variable, die ein **Resource** -Objekt darstellt.
