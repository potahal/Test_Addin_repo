
# AppointmentItem.OutlookInternalVersion Property (Outlook)

Gibt einen  **Long** -Buildnummer der Outlook-Anwendung für ein Outlook-Element zurück. Schreibgeschützt.


## Syntax

 _Ausdruck_. **OutlookInternalVersion**

 _Ausdruck_ Eine Variable, die ein **AppointmentItem** -Objekt darstellt.
