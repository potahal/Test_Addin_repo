
# MsoEnvelope-Objekt (Office)

Ermöglicht den Zugriff auf Funktionen, mit denen Sie Dokumente als E-Mail-Nachrichten direkt aus Microsoft Office-Anwendungen heraus versenden können.


## Bemerkungen

Verwenden Sie die  **MailEnvelope** -Eigenschaft des **Document** -Objekts, Objekts **Chart** oder **Worksheet** -Objekt (abhängig von der Anwendung, die Sie verwenden) um ein **MsoEnvelope** -Objekt zurückzugeben.


## Beispiel

Im folgenden Beispiel wird das aktive Microsoft Word-Dokument als E-Mail-Nachricht an die durch eine Unterroutine übergebene E-Mail-Adresse gesendet.


```
Sub SendMail(ByVal strRecipient As String) 
 
 'Use a With...End With block to reference the MsoEnvelope object. 
 With Application.ActiveDocument.MailEnvelope 
 
 'Add some introductory text before the body of the e-mail. 
 .Introduction = "Please read this and send me your comments." 
 
 'Return a Microsoft Outlook MailItem object that 
 'you can use to send the document. 
 With .Item 
 
 'All of the mail item settings are saved with the document. 
 'When you add a recipient to the Recipients collection 
 'or change other properties, these settings persist. 
 .Recipients.Add strRecipient 
 .Subject = "Here is the document." 
 
 'The body of this message will be 
 'the content of the active document. 
 .Send 
 End With 
 End With 
End Sub
```
