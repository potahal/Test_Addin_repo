
# RulerLevel2.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **RulerLevel2** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **RulerLevel2** -Objekt zurückgibt.


### Rückgabewert

Object
