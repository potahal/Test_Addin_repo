
# ReportItem.Read Event (Outlook)

Tritt auf, wenn eine Instanz des übergeordneten Objekts vom Benutzer zum Bearbeiten geöffnet wird.


## Syntax

 _Ausdruck_. **Read**

 _Ausdruck_ Eine Variable, die ein **ReportItem** -Objekt darstellt.


## Bemerkungen

Das  **Read** -Ereignis unterscheidet sich von der **[Open](f44fe7fe-29b3-f1ab-70ee-0e395ad6896a.md)** -Ereignis, dass **eintritt, wenn der Benutzer das Element in einer Ansicht auswählt, dass in der Zelle bearbeiten ermöglicht, und wenn das Element in einem  **[Inspektor](d7384756-669c-0549-1032-c3b864187994.md)** geöffnet wird**.
