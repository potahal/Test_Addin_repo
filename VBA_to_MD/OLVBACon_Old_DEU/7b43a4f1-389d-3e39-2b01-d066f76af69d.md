
# DistListItem.Saved Property (Outlook)

Gibt einen  **booleschen** Wert, der **True** ist, wenn das Outlook-Element nicht seit dem letzten Speichern geändert wurde. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Saved**

 _Ausdruck_ Eine Variable, die ein **DistListItem** -Objekt darstellt.
