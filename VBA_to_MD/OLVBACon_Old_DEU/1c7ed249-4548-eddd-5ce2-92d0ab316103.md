
# FormRegion.Form Property (Outlook)

Gibt ein  **Objekt** zurück, das **MSForms.UserForm** -Objekt darstellt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Form**

 _Ausdruck_ Eine Variable, die ein **FormRegion** -Objekt darstellt.
