
# Month Object (Project)

Stellt einen Monat in einem Jahr an. Das  **Month** -Objekt ist ein Mitglied der **[Months](5db0ed37-cc23-7bc8-ebe5-fdaf6275b5db.md)** -Auflistung.


## Beispiel

 **Verwenden des Month-Objekts**

Verwenden Sie  **Monate** ( _Index_ ), wobei _Index_ der Monatsindex, der Monatsname oder **PjMonth** -Konstante ist, um ein einzelnes **Month** -Objekt zur체ckzugeben. Im folgenden Beispiel wird ermittelt die Anzahl von Arbeitstagen in jeden Monat des 2012 f체r jede ausgew채hlte Ressource.




```
Dim R As Resource 
Dim D As Integer, M As Integer, WorkingDays As Integer 
 
For Each R In ActiveSelection.Resources() 
    WorkingDays = 0 

    With R.Calendar.Years(2012) 
        For M = 1 To .Months.Count 
            WorkingDays = 0 
            For D = 1 To .Months(M).Days.Count 
                If .Months(M).Days(D).Working = True Then 
                    WorkingDays = WorkingDays + 1 
                End If 
            Next D 

            MsgBox "There are " &amp; WorkingDays &amp; " working days in " &amp; _
                .Months(M).Name &amp; " for " &amp; R.Name &amp; "." 
        Next M 
    End With 
Next R
```

 **Verwenden der Months-Auflistung**

Verwenden Sie die  **[Months](615a4f5c-bda7-f684-1c29-d8003badf3a8.md)** -Eigenschaft, um eine **Months** -Auflistung zur체ckzugeben. Im folgenden Beispiel wird ermittelt die Anzahl der Monate in 2012.
