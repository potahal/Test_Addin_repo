
# ServerPolicy.Name-Eigenschaft (Office)

Ruft den Namen des  **ServerPolicy** -Objekts ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Name**

 _Ausdruck_ Ein Ausdruck, der ein **ServerPolicy** -Objekt zurückgibt.


### Rückgabewert

String
