
# ShapeRange.HorizontalFlip-Eigenschaft (Projekt)
Ruft einen Wert, der angibt, ob der Formenbereich an der horizontalen Achse gespiegelt wird. Read-only  **[MsoTriState](http://msdn.microsoft.com/en-us/library/office/ff860737%28v=office.15%29)**.

## Syntax

 _Ausdruck_. **HorizontalFlip**

 _Ausdruck_ Eine Variable, die ein ShapeRange **ShapeRange**-Objekt darstellt.


## Eigenschaftswert

 **MSOTRISTATE**
