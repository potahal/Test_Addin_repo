
# ContactItem.MiddleName Property (Outlook)

Gibt zurück oder legt eine  **Zeichenfolge** zur Darstellung des zweiten Vornamens für den Kontakt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **MiddleName**

 _Ausdruck_ Eine Variable, die ein **ContactItem** -Objekt darstellt.


## Bemerkungen

Diese Eigenschaft wird durch Analysieren aus der  **[FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)** -Eigenschaft, aber möglicherweise geändert oder unabhängig eingegeben werden, wenn Sie nicht richtig analysiert wurde. Beachten Sie, dass es sich bei einer solchen geändert oder Einträge für diese Eigenschaft von der Einträge in **FullName** nachfolgenden Änderungen überschrieben werden.
