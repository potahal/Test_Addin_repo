
# MetaProperty.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **MetaProperty** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **MetaProperty** -Objekt zurückgibt.


### Rückgabewert

Object
