
# CustomXMLPart.SchemaCollection-Eigenschaft (Office)

Dient zum Abrufen oder Festlegen eines  **CustomXMLSchemaCollection** -Objekts, das den Satz von an einen gebundenen Datenbereich in einem Dokument angefügten Schemas darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **SchemaCollection**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLPart** -Objekt zurückgibt.


### Rückgabewert

CustomXMLSchemaCollection
