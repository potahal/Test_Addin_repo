# EffectParameters.Item プロパティ (Office)




指定したインデックス位置または指定した一意の id 読み取り専用では、 **EffectParameter**オブジェクトを取得します。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Item**(**Index**)




            UNRESOLVED_TOKEN_VAL(offexpression)
            **EffectParameters** オブジェクトを返すオブジェクト式を指定します。

### パラメーター

|**名前**|**必須/オプション**|**データ型**|**説明**|
|---|---|---|---|
|*Index*|
                        UNRESOLVED_TOKEN_VAL(offrequired)
                      |**バリアント型 (Variant)**|いずれかを表す整数インデックスまたは**EffectParameter**の位置を表す文字列を指定します。|
