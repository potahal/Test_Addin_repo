# ServerPolicy.Id プロパティ (Office)




サーバー ポリシーの ID を取得します。値の取得のみ可能です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Id**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ServerPolicy** オブジェクトを表すオブジェクト式を指定します。

### 戻り値
文字列型 (String)
