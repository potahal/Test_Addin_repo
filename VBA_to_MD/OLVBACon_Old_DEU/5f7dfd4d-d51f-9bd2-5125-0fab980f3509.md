
# MeetingItem.Sensitivity Property (Outlook)

Gibt eine Konstante in der  **[OlSensitivity](611d23ca-40ee-17e9-2560-99c5508f6e29.md)** -Aufzählung zurück bzw. legt diese fest. Die Konstante gibt die Vertraulichkeit des Outlook-Elements an. Wert mit Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Sensitivity**

 _Ausdruck_ Eine Variable, die ein **MeetingItem** -Objekt darstellt.


## Bemerkungen

Diese Eigenschaft entspricht der MAPI-Eigenschaft  **PidTagSensitivity**.
