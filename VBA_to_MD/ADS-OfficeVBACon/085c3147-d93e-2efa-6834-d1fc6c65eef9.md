# Font2.NameAscii プロパティ (Office)




英数字 (文字コードが 0 (ゼロ) ～ 127 の文字) に使用するフォントを取得または設定します。値の取得および設定が可能です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**NameAscii**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Font2** オブジェクトを返す式。

### 戻り値
文字列型 (String)
