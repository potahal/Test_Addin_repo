
# CommandBarPopup.Delete-Methode (Office)

Löscht das  **CommandBarPopup** -Objekt aus der Auflistung.


## Syntax

 _Ausdruck_. **Delete**( ** _Temporary_** )

 _Ausdruck_ Erforderlich. Eine Variable, die ein **[CommandBarPopup](a8ae06a3-1d7b-a531-91df-756fafee5314.md)** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Temporary_|Optional|**Variant**|**True**, wenn das Steuerelement für die aktuelle Sitzung zu löschen. Das Steuerelement wird von die Anwendung in der nächsten Sitzung erneut angezeigt.|
