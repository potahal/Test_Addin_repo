
# MeetingItem.RTFBody Property (Outlook)

Zurück oder legt einen  **Byte** -Array, die den Textkörper des Elements Microsoft Outlook in Rich-Text-Format darstellt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **RTFBody**

 _Ausdruck_ Eine Variable, die ein **[MeetingItem](b75730f5-b395-3d66-5acd-b64fd8fcd78f.md)** -Objekt darstellt


## Hinweise

Sie können die  **StrConv** -Funktion in Microsoft Visual Basic für Applikationen (VBA) oder die Methode **System.Text.Encoding.AsciiEncoding.GetString()** in c# oder Visual Basic verwenden Sie zum Konvertieren eines Bytearrays in eine Zeichenfolge.
