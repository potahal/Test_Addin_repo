
# OlkTextBox.MaxLength Property (Outlook)

Zurückgeben oder Festlegen einer  **Long**, der die maximale Anzahl von Zeichen für den **[Wert](6c0efe12-7b85-344e-a14c-3e628b0a3877.md)** des Steuerelements angibt. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **MaxLength**

 _Ausdruck_ Eine Variable, die ein **OlkTextBox** -Objekt darstellt.


## Hinweise

Der Standardwert ist Null. Dies bedeutet lediglich, dass für dieses Steuerelement keine maximale Länge für den Wert definiert wurde. Basierend auf dem verfügbaren Arbeitsspeicher oder anderen Faktoren kann trotzdem eine maximale Länge gelten.
