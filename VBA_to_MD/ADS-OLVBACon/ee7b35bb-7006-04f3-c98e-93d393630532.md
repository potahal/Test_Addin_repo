# ContactItem.NetMeetingAlias プロパティ (Outlook)(機械翻訳)




ユーザーの Microsoft NetMeeting ID またはエイリアスを示す**文字列**を設定または返します。読み取り/書き込み。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**NetMeetingAlias**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ContactItem** オブジェクトを表す変数を指定します。
