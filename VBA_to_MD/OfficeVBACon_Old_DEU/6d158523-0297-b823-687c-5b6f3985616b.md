
# CustomXMLParts.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **CustomXMLParts** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLParts** -Objekt zurückgibt.


### Rückgabewert

Object
