
# CustomXMLPrefixMappings-Objekt (Office)

Stellt eine Auflistung von  **CustomXMLPrefixMapping** -Objekten dar.


## Beispiel

Das folgende Beispiel erstellt ein  **CustomXMLPrefixMapping** -Objekt mithilfe der **CustomXMLPrefixMapping** -Auflistung ein Namespace und ein Präfix hinzugefügt.


```
Dim objNamespace As CustomXMLPrefixMapping 
 
objNamespace = CustomXMLPrefixMappings.AddNamespace("xs", "urn:invoice:namespace")
```
