
# Application.EnterpriseProjectDelete メソッド (Project)

Project Server データベースからプロジェクトを削除します。Project Professional でのみ使用可能です。


## 構文

 _式_. **EnterpriseProjectDelete**( ** _Name_** )

 _式_ **Application** オブジェクトを表す変数です。


### パラメーター



|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _Name_|省略可能|**String**|Project Server データベースから削除されるプロジェクトの名前です。|

### 戻り値

 **ブール型 (Boolean)**
