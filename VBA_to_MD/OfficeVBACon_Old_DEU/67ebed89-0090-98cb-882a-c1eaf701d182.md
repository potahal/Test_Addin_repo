
# TextRange2.InsertAfter-Methode (Office)

Fügt Text rechts vom vorhandenen Text im  **TextRange2** -Objekt.


## Syntax

 _Ausdruck_. **InsertAfter**( ** _NewText_** )

 _Ausdruck_ Ein Ausdruck, der ein **TextRange2** -Objekt zurückgibt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _NewText_|Optional|**String**|Enthält den einzufügenden Text.|

### Rückgabewert

TextRange2
