# ResourceGroups2.Application プロパティ (Project)




**アプリケーション**オブジェクトを取得します。読み取り専用**アプリケーション**です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Application**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **ResourceGroups2** オブジェクトを返すオブジェクト式を指定します。

### 戻り値
Application
