
# MailItem.ReceivedByName Property (Outlook)

Gibt eine  **Zeichenfolge**, die den Anzeigenamen des tatsächlichen Empfängers der e-Mail-Nachricht darstellt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **ReceivedByName**

 _Ausdruck_ Eine Variable, die ein **MailItem** -Objekt darstellt.


## Bemerkungen

Diese Eigenschaft entspricht der MAPI-Eigenschaft  **PidTagReceivedByName**.
