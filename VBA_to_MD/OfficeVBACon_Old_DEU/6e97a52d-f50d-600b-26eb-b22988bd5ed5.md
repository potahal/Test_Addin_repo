
# CommandBarButton.HelpFile-Eigenschaft (Office)

Ruft ab oder legt den Dateinamen für das Hilfethema, das  **CommandBarButton** -Steuerelement zugeordnet ist. Lese-/Schreibzugriff.


## 


 **Hinweis**  "CommandBar"-Objekte wurden in einigen Microsoft Office-Anwendungen durch die neue Menüband-Komponente der Microsoft Office Fluent-Benutzeroberfläche abgelöst. Suchen Sie in der Hilfe nach dem Stichwort "Menüband", um weitere Informationen zu erhalten.


## Syntax

 _Ausdruck_. **HelpFile**

 _Ausdruck_ Eine Variable, die ein **CommandBarButton** -Objekt darstellt.


### Rückgabewert

String


## Bemerkungen

Zum Verwenden dieser Eigenschaft müssen Sie auch die  **HelpContextID** -Eigenschaft festlegen. Hilfethemen werden mit der Tastenkombination UMSCHALTTASTE+F1 aufgerufen.
