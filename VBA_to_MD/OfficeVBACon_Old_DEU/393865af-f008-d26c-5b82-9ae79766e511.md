
# ThemeFonts-Objekt (Office)

Stellt eine Auflistung von Haupt- und Hilfsschriftarten im Schriftartenschema eines Microsoft Office-Designs dar.


## Beispiel

Im folgende Beispiel wird ein  **ThemeFonts** -Objekt auf eine Design-Hilfsschriftart.


```
Dim tTheme As OfficeTheme 
Dim tfThemeFonts As ThemeFonts 
Set tfThemeFonts = tTheme.ThemeFontScheme.MinorFont 

```
