# CustomXMLNode.OwnerDocument プロパティ (Office)




指定したノードに関連付けられている Microsoft Excel ブック、Microsoft PowerPoint プレゼンテーション、または Microsoft Word 文書を表すオブジェクトを取得します。値の取得のみ可能です。

>[!NOTE]
>
              UNRESOLVED_TOKEN_VAL(osoffdevnodtdsincustomxml)
            





## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**OwnerDocument**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **CustomXMLNode** オブジェクトを表すオブジェクト式を指定します。

### 戻り値
オブジェクト型 (Object)
