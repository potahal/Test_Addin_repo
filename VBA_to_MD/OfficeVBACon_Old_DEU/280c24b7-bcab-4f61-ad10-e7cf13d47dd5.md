
# PolicyItem.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **PolicyItem** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **PolicyItem** -Objekt zurückgibt.


### Rückgabewert

Object
