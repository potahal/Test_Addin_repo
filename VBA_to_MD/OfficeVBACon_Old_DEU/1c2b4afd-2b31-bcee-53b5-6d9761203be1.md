
# CommandBarControls.Count-Eigenschaft (Office)

Ruft die Anzahl der Steuerelemente auf einer Befehlsleiste ab. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Ein Ausdruck, der ein **CommandBarControls** -Objekt zurückgibt.


### Rückgabewert

Integer
