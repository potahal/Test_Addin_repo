
# CodeMask.Parent Property (Project)

Das übergeordnete Element des  **CodeMask** -Objekts ab. Read-only **OutlineCode**.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Eine Variable, die ein **CodeMask** -Objekt darstellt.


## Bemerkungen

Verwenden Sie die  **Parent** -Eigenschaft, um die Eigenschaften oder Methoden des übergeordneten Elements eines Objekts zuzugreifen.
