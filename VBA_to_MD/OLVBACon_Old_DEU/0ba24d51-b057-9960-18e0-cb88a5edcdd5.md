
# Actions.Count Property (Outlook)

Gibt ein  **Long** zurück, der die Anzahl der Objekte in der angegebenen Auflistung angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ A variable that represents an **Actions** object.
