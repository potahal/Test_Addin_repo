
# WebPageFonts.Count-Eigenschaft (Office)

Ruft ein  **Long** zurück, der die Anzahl der Elemente in der **WebPageFonts** -Objekt angibt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Count**

 _Ausdruck_ Eine Variable, die ein **WebPageFonts** -Objekt darstellt.


### Rückgabewert

Long
