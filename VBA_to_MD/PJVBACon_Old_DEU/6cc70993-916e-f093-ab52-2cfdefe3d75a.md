
# Exception.Finish Property (Project)

Ruft den Endtermin einer Ausnahme ab, oder legt diesen fest.  **Variant** -Wert mit Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Finish**

 _Ausdruck_ Eine Variable, die ein **Exception** -Objekt darstellt.
