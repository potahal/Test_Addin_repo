
# Application.ConnectionStatusChanged イベント (Project)

Project Server との接続の状態が変更されたときに発生します。Project Professional でのみ使用できます。


## 構文

 _式_. **ConnectionStatusChanged**( ** _online_**, )

 _式_ **Application** オブジェクトを表す変数です。


### パラメーター



|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _online_|必須|**ブール型 (Boolean)**|の **場合は true。** Project Professional のプロジェクト サーバーに接続されている場合それ以外の場合、 **false を指定** します。|

### 戻り値

なし
