
# Font2.Caps-Eigenschaft (Office)

Ruft einen Wert ab oder legt einen solchen fest, der angibt, dass der Text in Großbuchstaben formatiert werden soll. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Caps**

 _Ausdruck_ Ein Ausdruck, der ein **Font2** -Objekt zurückgibt.


### Rückgabewert

MsoTextCaps
