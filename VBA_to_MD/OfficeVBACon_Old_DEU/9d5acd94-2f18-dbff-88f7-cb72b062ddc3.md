
# CustomXMLNode.Text-Eigenschaft (Office)

Ruft den Text für den aktuellen Knoten ab oder legt ihn fest. Lese-/Schreibzugriff.


## Syntax

 _Ausdruck_. **Text**

 _Ausdruck_ Ein Ausdruck, der ein **CustomXMLNode** -Objekt zurückgibt.


### Rückgabewert

String
