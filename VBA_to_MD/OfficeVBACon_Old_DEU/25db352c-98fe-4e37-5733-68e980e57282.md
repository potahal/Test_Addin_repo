
# ThemeFontScheme.Parent-Eigenschaft (Office)

Ruft ab, der  **übergeordnete** Objekt für das **ThemeFontScheme** -Objekt. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ Ein Ausdruck, der ein **ThemeFontScheme** -Objekt zurückgibt.


### Rückgabewert

Object
