
# Application.Reminders Property (Outlook)

Gibt eine  **[Reminders](66b94251-7fe4-886b-7c29-7feac4440dee.md)** -Auflistung zur端ck, die alle aktuellen Erinnerungen darstellt. Schreibgesch端tzt.


## Syntax

 _Ausdruck_. **Reminders**

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


## Beispiel

Im folgenden Beispiel wird die  **Reminders** -Auflistung zur端ckgegeben und der Titel aller Erinnerungen in der Auflistung angezeigt. Wenn keine aktuellen Erinnerungen verf端gbar sind, wird dem Benutzer eine Meldung angezeigt.


```
Sub ViewReminderInfo() 
 
 'Lists reminder caption information 
 
 Dim objRem As Outlook.Reminder 
 
 Dim objRems As Outlook.Reminders 
 
 Dim strTitle As String 
 
 Dim strReport As String 
 
 
 
 Set objRems = Application.Reminders 
 
 strTitle = "Current Reminders:" 
 
 strReport = "" 
 
 'If there are reminders, display message 
 
 If Application.Reminders.Count <> 0 Then 
 
 For Each objRem In objRems 
 
 'Add information to string 
 
 strReport = strReport &amp; objRem.Caption &amp; vbCr 
 
 Next objRem 
 
 'Display report in dialog 
 
 MsgBox strTitle &amp; vbCr &amp; vbCr &amp; strReport 
 
 Else 
 
 MsgBox "There are no reminders in the collection." 
 
 End If 
 
End Sub
```
