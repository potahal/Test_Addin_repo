
# OlkTextBox.DoubleClick Event (Outlook)

Tritt auf, wenn der Benutzer im Steuerelement doppelklickt.


## Syntax

 _Ausdruck_. **DoubleClick**

 _Ausdruck_ Eine Variable, die ein **OlkTextBox** -Objekt darstellt.
