
# Application.ProjectCalculate Event (Project)

Tritt auf, nachdem ein Projekt berechnet wurde.


## Syntax

 _Ausdruck_. **ProjectCalculate**( ** _pj_**, )

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _pj_|Erforderlich|**Project**|Das Projekt, das berechnet wurde.|

nothing


## Bemerkungen

 **Project** -Ereignisse treten nicht auf, wenn das Projekt in ein anderes Dokument oder eine andere Anwendung eingebettet ist.
