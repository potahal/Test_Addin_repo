
# MailItem.Save-Methode (Outlook)

Speichert das Microsoft Outlook-Element im aktuellen Ordner oder, falls es sich um ein neues Element handelt, im Outlook-Standardordner dieses Elementtyps.


## Syntax

 _Ausdruck_. **Save**

 _Ausdruck_ Eine Variable, die ein **MailItem** -Objekt darstellt.


## Hinweise

Wenn es sich bei einem E-Mail-Element um eine Inlineantwort handelt, löst das Aufrufen von  **Save** für dieses E-Mail-Element ggf. einen Fehler und unerwartetes Verhalten aus.
