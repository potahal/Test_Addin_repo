
# Application.ColumnEdit Method (Project)

Zeigt das Dialogfeld  **Definition Spalte** für die angegebene Spalte an.


## Syntax

 _Ausdruck_. **ColumnEdit**( ** _Column_** )

 _Ausdruck_ Eine Variable, die ein **Application** -Objekt darstellt.


### Parameter



|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Column_|Optional|**Integer**|Die Nummer der zu ändernden Spalte. Der Standardwert ist die Nummer der aktiven Spalte.|

### Rückgabewert

 **Boolean**
