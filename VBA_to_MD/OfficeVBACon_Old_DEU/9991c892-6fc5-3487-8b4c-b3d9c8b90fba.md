
# Font2.Application-Eigenschaft (Office)

Ruft ein Objekt, das die Anwendung darstellt, der in das  **Font2** -Objekt verwendet wird. Schreibgeschützt.


## Syntax

 _Ausdruck_. **Application**

 _Ausdruck_ Ein Ausdruck, der ein **Font2** -Objekt zurückgibt.


### Rückgabewert

Object
