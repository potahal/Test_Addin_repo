
# CardView.Apply Method (Outlook)

Wendet das  **[CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** -Objekt auf die aktuelle Ansicht an.


## Syntax

 _Ausdruck_. **Apply**

 _Ausdruck_ Eine Variable, die ein **CardView** -Objekt darstellt.
