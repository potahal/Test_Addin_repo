
# GroupCriterion2.FontColor Property (Project)

Ruft ab oder legt die Farbe der Schriftart für ein Feld als ein Kriterium in einer Gruppendefinition verwendet wird. Lese-/Schreibzugriff  **PjColor**.


## Syntax

 _Ausdruck_. **FontColor**

 _Ausdruck_ Ein Ausdruck, der ein **GroupCriterion2** -Objekt zurückgibt.


## Bemerkungen

Die  **FontColor** -Eigenschaft kann eine der folgenden **[PjColor](46108cf5-1e35-9774-b424-6c84223d9aac.md)** -Konstanten sein:


|||
|:-----|:-----|
|**pjColorAutomatic**|**pjNavy**|
|**pjAqua**|**pjOlive**|
|**pjBlack**|**pjPurple**|
|**pjBlue**|**pjRed**|
|**pjFuchsia**|**pjSilver**|
|**pjGray**|**pjTeal**|
|**pjGreen**|**pjYellow**|
|**pjLime**|**pjWhite**|
|**pjMaroon**||
Informationen zum Verwenden von RGB-Hexadezimalwerten finden Sie unter der  **[FontColorEx](5556ce54-315e-f676-6771-baec87853d8a.md)** -Eigenschaft.
