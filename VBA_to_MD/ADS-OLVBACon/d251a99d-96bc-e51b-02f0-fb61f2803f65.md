# Conversation.Application プロパティ (Outlook)(機械翻訳)




**Conversation** オブジェクトの親 Microsoft Outlook アプリケーションを表す **Application** オブジェクトを取得します。値の取得のみ可能です。

## 構文

            UNRESOLVED_TOKEN_VAL(offexpression).**Application**




            UNRESOLVED_TOKEN_VAL(offexpression)
            **Conversation** オブジェクトを表す変数を指定します。
