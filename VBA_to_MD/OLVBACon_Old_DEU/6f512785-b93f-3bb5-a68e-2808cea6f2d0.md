
# OlkCheckBox.Click Event (Outlook)

Tritt auf, wenn der Benutzer in das Steuerelement klickt.


## Syntax

 _Ausdruck_. **Click**

 _Ausdruck_ Eine Variable, die ein **OlkCheckBox** -Objekt darstellt.
